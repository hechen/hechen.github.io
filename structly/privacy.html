What Structly Does

Structly is a Safari extension that detects and formats JSON content in your browser. When you visit a page containing raw JSON, Structly renders it with syntax highlighting, collapsible tree views, and other tools to make JSON easier to read.

All processing is performed entirely on your device using JavaScript within Safari's extension sandbox.

Data We Collect

None.

Structly does not collect any personal information, usage data, analytics, diagnostics, or any other data of any kind. Specifically:

  • No personal information (name, email, identifiers)
  • No usage or analytics data
  • No browsing history or page content
  • No crash reports or diagnostics
  • No cookies or tracking technologies
  • No advertising identifiers

Data Sharing

We do not share any data with third parties because we do not have any data to share.

Network Requests

Structly makes zero network requests. It does not communicate with any server, API, or external service. The extension operates entirely offline within Safari.

Permissions

Structly requests the "activeTab" permission, which allows it to read the content of the current page to detect and format JSON. This permission is scoped to the active tab and does not grant access to browsing history, other tabs, or any data beyond the current page content.

Data Storage

Structly stores your theme preference locally on your device using Safari's built-in extension storage. This data never leaves your device and can be cleared by removing the extension.

Children's Privacy

Structly does not collect information from anyone, including children under 13.

Changes to This Policy

If we ever change this policy, we will update this page with the new date. Given that we collect no data, we don't anticipate meaningful changes.
